# Checkers
A classic board game implementation built with interactive visuals, intelligent gameplay, and user-friendly controls. Players can challenge each other locally or test their skills against the computer.

---

## Features & Functionalities

### Gameplay Modes
- **Player vs Player (PVP):** Two players can compete on the same device.  
- **Player vs Computer (PVC):** Play against an AI opponent that makes valid moves automatically.

### Game Mechanics
- Players make moves using **mouse clicks**.  
- **Turns alternate** automatically between players.  
- The game validates all moves (**invalid moves are disallowed** and show a visual warning).  
- **Game-ending conditions** are detected:
  - No valid moves left.
  - No remaining pieces.

### Visuals & Interaction
- **Real-time board updates** after each move.  
- **Turn indicators**, **invalid move warnings**, and **winner notifications** displayed on-screen.  
- Option to **pause** or **return to the main menu** anytime.

### Settings & Customization
- Change **background color** and **toggle sound effects**.  
- Track **lowest time to win** across sessions.  
- **Save and load** game progress.

### Game Management
- **Start a new game** or **restart** after a match ends.  
- Return to the **main menu** to adjust settings or switch modes.
